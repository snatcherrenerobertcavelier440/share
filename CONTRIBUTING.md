# Contributing

## Development

```sh
bun install
bun run check
```

Use Cloudflare Turnstile test keys for local browser development as shown in the README. Do not add authentication bypasses, plaintext capability storage, or public R2 delivery paths.

## Pull requests

Keep changes narrow, update tests for behavioral changes, and update the relevant README tutorial, how-to, explanation, or reference section when the user workflow or public safety model changes.
